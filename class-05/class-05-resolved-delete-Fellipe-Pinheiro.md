## Realizar 3 counts na collection pokemons.##
```
// .count -- HP maior que 100
5e14de9d5d4e(mongod-3.0.7) be-mean> db.pokemons.count({ hp: {$gt: 100} } )
48

// .count -- Attack AND Defense maior igual a 100
5e14de9d5d4e(mongod-3.0.7) be-mean> db.pokemons.count( {$and: [{attack: {$gte: 100} }, {defense: {$gte: 100} }] } )
45

// .count -- Speed diferente de 80
5e14de9d5d4e(mongod-3.0.7) be-mean> db.pokemons.count( {speed: {$ne: 80} } )
584

```
