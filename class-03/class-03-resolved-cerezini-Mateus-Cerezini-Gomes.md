## Liste todos Pokemons com a altura **menor ou igual que** 0.5 **E** do tipo grama
```
var query = {$and: [{height: {$lte: 0.5}},{type:'grama'}]}
matrix(mongod-3.0.7) be-mean-instagram> query
{
  "$and": [
    {
      "height": {
        "$lte": 0.5
      }
    },
    {
      "type": "grama"
    }
  ]
}
```
