## Liste todos Pokemons com name 'Pikachu' OU com attack menor ou igual que 0.5 (passo 4)

```
> var query = {$or: [{ name: 'Pikachu'}, { attack: {$lte: 0.5}} ]}
> db.pokemons.find(query)
>
```
