## 3- lista todos pokemons com altura menor ou igual a 1.9 e do tipo eletrico;
´´´
> query = {$and : [{height : {$lte: 1.9}},{name : 'Raikou'}]}
{
        "$and" : [
                {
                        "height" : {
                                "$lte" : 1.9
                        }
                },
                {
                        "name" : "Raikou"
                }
        ]
}
